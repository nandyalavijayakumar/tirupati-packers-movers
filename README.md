# 🚚 Tirupati Packers & Movers

A modern responsive website built using **Next.js** for a Packers & Movers business in Tirupati.

---

## 🚀 Tech Stack

- Next.js 14+
- React
- Tailwind CSS (or CSS Modules if using old version)
- JavaScript (ES6+)

---

## 📦 Features

- Responsive UI for all devices
- WhatsApp booking integration
- Service listing (House, Office, Vehicle shifting)
- Trust & testimonial sections
- Contact & CTA sections
- SEO-friendly structure

---

## 🖥️ Pages

- Home
- Services
- Booking
- Contact
- About
